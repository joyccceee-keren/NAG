```

7. Wait for "Success" message
8. Go back to Step 3 and order again
```

## ⚠️ IF SOMETHING GOES WRONG

**Problem: Server not running**
- Solution: In terminal, type: `npm start`

**Problem: WhatsApp page shows error**
- Solution: Refresh page (F5), wait 5 seconds

**Problem: QR code won't scan**
- Solution: Move closer to computer screen, keep still

**Problem: "Can't reach this page"**
- Solution: Server crashed. Type in terminal: `npm start`
